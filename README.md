## Overview

The Database Migration Utility project provides a seamless way to manage database schema versions leveraging PostgreSQL (other databases will be added later). Through a CLI interface, users can execute up or down migrations. The software's core functionality lies in applying migrations safely, handling database connections, and dynamically loading the database driver as needed.
